# 레드 블랙베리 트리

레드 블랙 트리 알고리즘 기반 노드 탐색 프로그램

## 기능
- key,value 형식의 파일 -> 레드 블랙 트리 자료구조로 메모리 적재
- 전체 검색
- 부분 트리 검색
- 데이터 추가
- pre-order 순회 결과 출력
- in-order 순회 결과 출력
- post-order 순회 결과 출력
- level-order 순회 결과 출력

## 빌드 및 실행
```sh
gcc fileToKeyValue.c -o test
./text
```
